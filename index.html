Zombie Dash

A Survival & Rescue Game

I built a multi-level, run-and-gun game in C++ that sets Player Penelope on a thrilling rescue mission in a zombie-infested building. Added features include multiple levels of zombie intelligence and citizen fear, nostalgic 2D graphics and startling audio.

Pulse Oximeter

An Oxygen Measuring Device

I built the hardware and developed the software of a biomedical device that measures and translates IR light to the user’s pulse. Added features include an Arduino-based OLED screen to visualize the user’s oxygen levels.
